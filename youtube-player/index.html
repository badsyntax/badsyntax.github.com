jQuery Youtube Player

The jQuery Youtube Player plugin builds a custom video player, toolbar and playlist area. It uses the Youtube Javascript API which is available through the plugin API. The plugin is easily customizable and easy to hook into using the many options and event functions.

Examples:

  • Basic
  • Advanced
  • Multiple Playlists
  • Custom Playlist
  • Positioning
  • Themes
