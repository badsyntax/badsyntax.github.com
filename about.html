About this site

I built this microsite to showcase some of my projects hosted on Github.

The home page of this site gives an overview of my github profile and activity. I set up a very simple REST service using Kohana and a PHP library which uses the Github API to retrieve data from Github. The results are retrieved cross-domain from my server at home ala JSONP.

I use my simple jQuery Render plugin to render the data rows in a HTML template, and use my jQuery Scrollbar plugin to scroll the commits container.

The site is coded using HTML5 markup, taking advantage of the radness that is CSS3. It probably won't look very good in older browsers. I'm using Remy Sharp's HTML5 enabling script to ensure the older browsers acknowledge the new HTML5 tags.
